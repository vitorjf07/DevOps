# Instruções

1. Abrir o projeto no VS Code.
2. Instalar as dependencias do projeto.

   ```bash
      cd aluno_online
      npm install
   ```

3. Executar o projeto no NodeJS.

   ```bash
      npm start
   ```

4. Abrir url <http://localhost:3000/> no navegador.

5. Autenticar com a matrícula 2020100 e senha senha123.
